# ODDS work logbook

## Initialize logbook

```
worklog init
```

## Sites

### Add a site

```
worklog site add <siteName>
```

### Remove a site

```
worklog site rm <siteName>
```

## Check-in

## Check-in to default site

```
worklog in
```

### Check-in to a specific site

```
worklog in <siteName>
```

## Stats

```
worklog stats
```

## Log

```
worklog log
```
